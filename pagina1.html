    HTML i +

    HTML significa HyperText Markup Language (lenguaje de marcado de hipertexto) . Es el lenguaje estándar que se utiliza para crear y estructurar contenido en la web.

  • HTML es un lenguaje de marcado, no un lenguaje de programación, lo que significa que anota el texto para definir cómo se estructura y se muestra en los navegadores web.
  • Constituye los componentes básicos de todos los sitios web y se complementa con CSS para el estilo y JavaScript para la interactividad.
