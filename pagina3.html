    Que es CSS?

    CSS (Cascading Style Sheets) es un lenguaje de programación que se utiliza para definir el diseño de páginas web. En español, se le conoce como Hojas de Estilo en Cascada.

    ¿Qué hace CSS?

  • Permite controlar el color, la fuente, la disposición de los elementos, las animaciones y los efectos visuales
  • Separa el contenido del diseño, lo que facilita la gestión del aspecto visual de las páginas web
  • Permite que varias páginas compartan la misma hoja de estilo

    • ¿Cómo se aplica CSS?

  • Se pueden utilizar hojas de estilos externas, internas o en línea.
  • Los estilos en línea se aplican directamente sobre elementos individuales a través del atributo style de CSS
